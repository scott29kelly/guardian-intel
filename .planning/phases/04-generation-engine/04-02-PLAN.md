  <action>
    Create `src/features/infographic-generator/services/infographicGenerator.ts`:

    1. **Progress callback type:**
       ```typescript
       type ProgressCallback = (progress: GenerationProgress) => void;
       ```

    2. **Module resolution helper (private):**
       `resolveModules(request: InfographicRequest): Promise<{ modules: TopicModule[]; audience: InfographicAudience }>`
       - If `mode === 'preset'`: look up preset via `getPresetById(request.presetId!)`, extract enabled modules from `preset.modules.filter(m => m.enabled).map(m => m.module)`, use preset.audience
       - If `mode === 'custom'`: use `request.selectedModules!` directly, use `request.audience || 'internal'`
       - If `mode === 'conversational'`: call `parseIntent(request.conversationalPrompt!)`, use returned modules and audience
       - Throw Error if preset not found or no modules resolved

    3. **Single model generation helper (private):**
       `generateSingle(prompt: string, scoringResult: ScoringResult, onProgress: ProgressCallback): Promise<ImageGenerationResponse>`
       - Build `ImageGenerationRequest` with prompt
       - If `scoringResult.selectedModel?.webSearchGrounding` is true, set `searchTypes: ['web', 'image']`
       - Set `resolution` from `scoringResult.selectedModel?.maxResolution` (cast to the union type)
       - Call `getAIRouter().generateImage(request, scoringResult.selectedModel!.id)`
       - Return the response

    4. **Chain generation helper (private):**
       `generateChain(modules: TopicModule[], data: Record<string, unknown>, audience: InfographicAudience, scoringResult: ScoringResult, onProgress: ProgressCallback): Promise<ImageGenerationResponse>`
       - Call `composeChainPrompts({ modules, data, audience, scoringResult })`
       - **Step 1 (accuracy):** emit progress `{ phase: 'generating', percent: 40, statusMessage: 'Generating your briefing...' }`
         - Build ImageGenerationRequest with accuracyPrompt, searchTypes if step[0].searchEnabled, resolution from chain step[0]
         - Call `getAIRouter().generateImage(request, chain.steps[0].model as AIModel)`
       - **Step 2 (refinement):** emit progress `{ phase: 'refining', percent: 70, statusMessage: 'Polishing final output...' }`
         - Build ImageGenerationRequest with refinementPrompt, set `referenceImages: [{ data: step1Response.imageData, mimeType: 'image/png' }]`, resolution from chain step[1]
         - Call `getAIRouter().generateImage(request, chain.steps[1].model as AIModel)`
       - Return the step 2 response (with `model` set to chain step[1] model string)

    5. **Main entry point:**
       ```typescript
       export async function generateInfographic(
         request: InfographicRequest,
         onProgress?: ProgressCallback
       ): Promise<InfographicResponse>
       ```
       - Default onProgress to no-op: `const progress = onProgress || (() => {})`
       - Record `startTime = Date.now()`
       - **Step 1 - Resolve modules:** emit `{ phase: 'data', percent: 5, statusMessage: 'Assembling customer data...' }`, call `resolveModules(request)`
       - **Step 2 - Assemble data:** emit `{ phase: 'data', percent: 15, statusMessage: hasWebSearchModules ? 'Pulling live weather conditions...' : 'Assembling customer data...' }`, call `assembleDataForModules(request.customerId, modules)`
       - **Step 3 - Score model:** emit `{ phase: 'scoring', percent: 25, statusMessage: 'Preparing generation...' }`, call `scoreRequest(request, modules)` (note: this is sync, not async)
       - **Step 4 - Compose prompt:** call `composePrompt({ data, modules, audience, scoringResult })`
       - **Step 5 - Generate:**
         - If `scoringResult.selectedChain`: call `generateChain(modules, data, audience, scoringResult, progress)`
         - Else: emit `{ phase: 'generating', percent: 40, statusMessage: 'Generating your briefing...' }`, call `generateSingle(prompt, scoringResult, progress)`
       - **Step 6 - Complete:** emit `{ phase: 'complete', percent: 100, statusMessage: 'Your briefing is ready!' }`
       - Return `InfographicResponse`:
         ```typescript
         {
           imageData: generationResult.imageData,
           model: generationResult.model,
           chainUsed: !!scoringResult.selectedChain,
           generationTimeMs: Date.now() - startTime,
           cached: false,
         }
         ```

    6. **Error handling:** Wrap the entire pipeline in try/catch. On error, emit `{ phase: 'complete', percent: 0, statusMessage: 'Generation failed' }` and rethrow.

    Do NOT implement retry logic in this service -- the AI Router adapter already has retry built in. Do NOT implement caching here -- the API route layer (Plan 03) handles cache check/store.
