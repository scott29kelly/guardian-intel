     f. Clear progress timers in finally block.

Create useInfographicBatch hook in src/features/infographic-generator/hooks/useInfographicBatch.ts:

1. Add "use client" directive.
2. Import useState, useCallback, useRef, useEffect from React.
3. Import InfographicResponse from "../types/infographic.types".
4. Define BatchCustomerStatus type: { customerId: string, status: 'pending' | 'generating' | 'complete' | 'error', result?: InfographicResponse, error?: string }
5. Return type: { isBatching: boolean, batchProgress: BatchCustomerStatus[], startBatch: (customerIds: string[]) => Promise<void>, cancelBatch: () => void }
6. Implementation:
   - useState for isBatching (boolean), batchProgress (BatchCustomerStatus[]).
   - useRef for polling interval ID and jobId.
   - startBatch(customerIds):
     a. Initialize batchProgress with all customerIds set to 'pending'.
     b. Set isBatching=true.
     c. POST to '/api/ai/generate-infographic/batch' with body { customerIds, autoSelectPresets: true }.
     d. Extract jobId from response.
     e. Start polling interval (every 2 seconds) via setInterval.
   - Polling logic:
     a. GET '/api/ai/generate-infographic/batch/{jobId}'.
     b. Map response results array to BatchCustomerStatus[].
     c. Update batchProgress state with latest results (progressive -- each customer appears as it completes).
     d. If response.status === 'complete', clear interval and set isBatching=false.
   - cancelBatch():
     a. Clear polling interval.
     b. Set isBatching=false.
   - useEffect cleanup: clear interval on unmount.
