Create src/features/infographic-generator/utils/offlineSupport.ts:

1. NO "use client" directive (this is a utility, not a component).
2. Constants:
   ```typescript
   const INFOGRAPHIC_CACHE_NAME = "infographic-images-v1";
   const MAX_CACHED_ITEMS = 50;
   ```

3. Function: cacheInfographic(customerId: string, presetId: string, imageData: string): Promise<void>
   - Opens cache via caches.open(INFOGRAPHIC_CACHE_NAME)
   - Creates a synthetic URL key: `/infographics/cache/${customerId}/${presetId}`
   - Converts base64 imageData to Blob (atob, Uint8Array, new Blob with image/png type)
   - Creates a Response from the Blob with headers (Content-Type: image/png, X-Cached-At timestamp)
   - Puts into cache: cache.put(url, response)
   - Prune old entries if cache exceeds MAX_CACHED_ITEMS (delete oldest by X-Cached-At header)

4. Function: getCachedInfographic(customerId: string, presetId: string): Promise<string | null>
   - Opens cache, tries to match the synthetic URL
   - If found, reads response as Blob, converts back to base64 data URL
   - Returns base64 string or null if not cached

5. Function: getCachedInfographicsForCustomer(customerId: string): Promise<Array<{ presetId: string, cachedAt: string }>>
   - Opens cache, gets all keys
   - Filters keys matching `/infographics/cache/${customerId}/`
   - Returns array of { presetId, cachedAt } from headers

6. Function: clearCachedInfographics(customerId?: string): Promise<void>
   - If customerId provided, delete only that customer's entries
   - If no customerId, delete entire cache

7. Function: isOnline(): boolean
   - Returns navigator.onLine

8. Function: onConnectivityChange(callback: (online: boolean) => void): () => void
   - Adds event listeners for 'online' and 'offline' events on window
   - Returns cleanup function that removes listeners

9. Export all functions as named exports.

Update public/sw.js:

  - Pattern: /\/infographics\/cache\/.*/
  - Strategy: CacheFirst with INFOGRAPHIC_CACHE_NAME
  - This ensures the service worker can serve cached infographics when offline
- Add this AFTER the existing registerRoute calls
- Keep the minified style consistent with the rest of the file
