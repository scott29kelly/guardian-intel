5. Component implementation (`export function AudioBriefingPlayer`):

   a. State:
      - `isPlaying: boolean` (default false)
      - `currentTime: number` (default 0)
      - `duration: number` (default 0)
      - `showShareSheet: boolean` (default false)
   
   b. Ref: `audioRef = useRef<HTMLAudioElement>(null)`

   c. Event handlers:
      - `handlePlayPause`: toggles play/pause. CRITICAL: only call `audioRef.current.play()` inside this direct click handler (per Research pitfall 2 -- mobile autoplay restrictions). On play, call `.play()`. On pause, call `.pause()`.
      - `handleTimeUpdate`: reads `audioRef.current.currentTime` and sets `currentTime` state.
      - `handleLoadedMetadata`: reads `audioRef.current.duration` and sets `duration` state.
      - `handleEnded`: sets `isPlaying` to false.
      - `handleSeek(e: React.MouseEvent<HTMLDivElement>)`: calculates click position relative to the seek bar container width, computes target time as `(clickX / containerWidth) * duration`, sets `audioRef.current.currentTime = targetTime`.

   d. Helper: `formatTime(seconds: number): string` returns "M:SS" format (e.g., "3:45").

   e. JSX layout (per D-05 -- sized for mobile thumb use, Navy/Gold/Teal):
      ```
      <div className="..." (outer container with Navy background, rounded, padding)>
        <!-- Hidden audio element -->
        <audio ref={audioRef} src={audioUrl} preload="metadata"
               onTimeUpdate={handleTimeUpdate}
               onLoadedMetadata={handleLoadedMetadata}
               onEnded={handleEnded}
               onPlay={() => setIsPlaying(true)}
               onPause={() => setIsPlaying(false)} />
        
        <!-- Controls row: Play/Pause + Time + Share -->
        <div className="flex items-center gap-3">
          <!-- Play/Pause button: 44x44px min touch target per mobile UX -->
          <button onClick={handlePlayPause}
                  className="w-11 h-11 flex items-center justify-center rounded-full"
                  style={{ backgroundColor: "#D4A656" }} // Gold
                  aria-label={isPlaying ? "Pause" : "Play"}>
            {isPlaying ? <Pause className="w-5 h-5 text-white" /> : <Play className="w-5 h-5 text-white ml-0.5" />}
          </button>
          
          <!-- Time display -->
          <span className="text-xs font-mono text-white/80 min-w-[70px]">
            {formatTime(currentTime)} / {formatTime(duration)}
          </span>

          <!-- Share button per D-07 -->
          <button onClick={() => setShowShareSheet(true)}
                  className="ml-auto w-9 h-9 flex items-center justify-center rounded-full hover:bg-white/10"
                  aria-label="Share audio briefing">
            <Share2 className="w-4 h-4 text-white/70" />
          </button>
        </div>

        <!-- Seek bar: Navy track with Gold fill -->
        <div className="mt-2 h-2 rounded-full cursor-pointer"
             style={{ backgroundColor: "rgba(255,255,255,0.15)" }}
             onClick={handleSeek}>
          <div className="h-full rounded-full transition-all"
               style={{ width: `${duration > 0 ? (currentTime / duration) * 100 : 0}%`, backgroundColor: "#D4A656" }} />
        </div>
      </div>
      ```

      Outer container background: `style={{ backgroundColor: "#1E3A5F" }}` (Navy per D-05).
      The component renders inline in the audio artifact card per D-06 -- no modal/bottom-sheet.

   f. ShareSheet rendered conditionally:
      ```tsx
      <ShareSheet
        isOpen={showShareSheet}
        onClose={() => setShowShareSheet(false)}
        imageData=""
        imageUrl={audioUrl}
        customerName={customerName}
      />
      ```
      Note: `imageData=""` because this is audio, not an image. ShareSheet uses `imageUrl` for the share link.
